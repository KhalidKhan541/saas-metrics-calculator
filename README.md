# SaaS Metrics Calculator

Calculate key SaaS metrics: MRR, ARR, churn rate, LTV, CAC, and more.

## Features
- MRR/ARR calculations
- Churn rate analysis
- LTV/CAC ratio
- Revenue forecasting
- Export reports
- Mobile-friendly

## Tech Stack
- HTML/CSS/JavaScript
- Cloudflare Pages

## Live Demo
https://saas-metrics-calculator.pages.dev

## Run Locally
Open index.html in your browser.
